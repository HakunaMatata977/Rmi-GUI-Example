# RMI String Operations

A Java RMI (Remote Method Invocation) example with a Swing GUI client.

## Features
- Convert string to Lowercase
- Convert string to Uppercase
- Check if string is Palindrome

## Files
- `RemoteMethod.java` - Remote interface
- `RemoteMethodImpl.java` - Server side implementation
- `RmiServer.java` - Server
- `RmiClient.java` - GUI Client

## How to Run
1. Compile all files: `javac *.java`
2. Run server: `java RmiServer`
3. Run client: `java RmiClient`

## Requirements
- Java JDK 8 or above
- IntelliJ IDEA (optional)
